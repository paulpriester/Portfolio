    Job board is a website built I for a non profit organization The Knowledge House. It’s dual-facing, where administrators can post jobs and students can view matches. To build it, I used React.js for frontend and Node.js,Express and MongoDB for backend. What I really liked about it is that it allowed me to learn about React deeper by breaking and fixing hundreds lines of code and MongoDB which I didn’t have much exposure to before. The project is also meaningful because I have benefitted from the jobs posted by The Knowledge House, and building a more advanced tool for the future will help other students like me start their careers in technology.

    CrytoWatch

    Built With

  • React

    • Description

    Cryptowatch is a site I constructed as a freelance project because of my interest in cryptocurrency . The website allows users to see the latest information on cryptocurrency. You can search up specific coins giving you the relevant data needed to day trade or look up news articles related to a specific coin. One of the favorite problem i solved on this project was small aesthetic feature. I wanted the percent change on the 24hr data to change colors depending if it was a positive or negative percent. To solve this I leaned the beauty of using ternary operators, this simplify parts of code for this particular problem and future problems.
